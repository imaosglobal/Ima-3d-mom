# אמא תלת־ממדית – Ima 3D

אתר הדגמה תלת־ממדית של דמות אמא בעברית, מבוסס שלוש.js.

## איך עובדים עם זה – How To Use

1. מצאו מודל תלת־ממד חינמי (`mom.glb`) והכניסו בתיקיית הרפוזיטורי.
2. יש לשים את כל הקבצים (כולל mom.glb ו-favicon.png) בתיקיה של הרפוזיטורי ולדחוס ל-ZIP לפני העלאה אם רוצים.
3. אפשר להעלות ל-GitHub ו/או לפרסם באמצעות Netlify או GitHub Pages.

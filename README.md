# Ima-3d-mom

## תיאור  
פרויקט זה מציג מודל תלת-ממד באמצעות קבצי glb.

## מה נדרש?
1. תיקיית `assets` המכילה קובץ glb של המודל שלכם:
   - לדוגמה: `assets/model.glb`
2. קובץ glb ניתן ליצור או להמיר דרך Blender או כל מחולל 3D אחר.

## איך מוסיפים מודל משלך?
- שמור את המודל בתיקייה `assets` בשם `model.glb` (או עדכן בשם בקוד).
- ודא שהקובץ בפורמט תקני ולא ריק.

## טעינת מודל
הקוד ב-`script.js` נטען את המודל מתוך `assets/model.glb`.  
במידה ואין קובץ glb תופיע הודעת שגיאה בדפדפן.

## הפנייה לקוד
לא לשכוח לעדכן את שם המודל בקוד אם משנים מ-model.glb!
